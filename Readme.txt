Install Python3
Then pip3 install -r requirements.txt

The code for the questions are all in Questions.py
Command to run Questions.py: python3 Questions.py

The program will display the following:

For Question 1, it will display a list of subway routes long names.

For Question 2, it will display the routes with the most and least stops along with their number of stops 
and a list of the stops that connect two or more subway routes along with a list of the relevant route names for
each of those stops.

For Question 3, after you input the start and end stops, it will display a rail route that connects the start and end stops.

For testing: 
File Test.py has the tests to test the functions in Questions.py
Command to run the test file: python3 Test.py
